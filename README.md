# Verkkotelakka Demo

Tee web-sovellus, jolla käyttäjä voi hakea oheisesta annetusta Sqlite-tietokannasta domaineja ja katsoa löytyneistä riveistä tietokannassa olevat kentät.

Voit valita itse ohjelmointikielen ja tarvittavat kirjastot / frameworkit.

Noudata hyviä ohjelmistokehityksen käytäntöjä. Tehtävän palautus ohjelmakoodina.

Lisäksi vastaa näihin kysymyksiin:

    Miksi uskot ohjelmasi toimivan oikein?
    Mikä sai valitsemaan käyttämäsi kielen ja kirjastot?
